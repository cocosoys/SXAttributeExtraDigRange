SX-Attribute插件的属性扩展，提供一个属性:DigRange
可使得手中物品能够范围破坏方块
配置文件:
```yaml
tool:
  # 允许触发范围破坏的物品
  pickaxe:
    - 278
    - 257
    - 270
    - 274
    - 285
# 允许被范围破坏的方块,支持 xx:xx 写法,例如 16:2
whiteblock:
  pickaxe:
    - "1"
    - "4"
    - "14"
    - "15"
    - "16"
    - "21"
    - "56"
    - "87"
    - "129"
    - "153"
```
